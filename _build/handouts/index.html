Sentiment Visualization

Introduction

In this exercise, you’ll continue solidifying your visualization skills using Plotly.

Research

Before you begin this exercise, you will want to ensure that you have a clear understanding of the different types of data visualization that exist, and what types of relationships / conclusions that you can draw from each. As a data analyst, you will have to be extremely knowledgable and comfortable making decisions about which chart/graph to use for different datasets. The ability to make good visualization decisions requires knowing your options for graphs/charts, as well as being inspired by visualizations you have seen. Thankfully, there are so many engaging visualization aggregators and examples on the internet.

During this part of the exercise, you’ll spend time generating a library of visualizations that inspire, impress, excite, or educate you. In your future endeavors as a data analyst, you can draw from this set of examples to help drive you towards a visualization that accomplishes your goals.

Create a Google Doc to house your research findings for this part of the exercise.

Do a search for each of the following chart/graph types, and choose 1-2 examples to save (screenshot or download an image of the visualization) and include in the Google Doc you create. Below each image, for each chart type answer the following questions:

  1. What is the definition of the chart? If it’s similar to other chart types, how do you differentiate this type from others?

  2. What is the key takeaway for this visualization? What is the central message it is communicating?

  3. What are the visual elements that make this visualization effective? Why does it stand out to you as a good example of this chart? (Color? Shapes? Size? Scale? Labels? Smart filtering choices? Something else?)

Setup

Make account on data.world.

Download two datasets as CSV files.

  • Airline sentiment dataset

  • Self-driving cars sentiment dataset

Make a virtualenv and install jupyter notebook, pandas, and plotly.

$ virtualenv env
$ source env/bin/activate
$ (env) pip3 install notebook
$ (env) pip3 install plotly
$ (env) pip3 install pandas

Now, start your Jupyter notebook server.

$ (env) jupyter notebook

This should open your browser to your jupyter notebook tree, but if it does not, open this link: http://localhost:8888/tree.

Lastly, create a new Jupyter Notebook by clicking the dropdown on the upper right that says “New” and then select “Python 3 (pykernal)”.

Airline Sentiment Data

Starting with the airline sentiment dataset, create the following charts using Plotly.

Conduct some initial data exploration to become familiar with the dataset.

Then, begin visualizing the data in Jupyter notebook using Plotly.

Visualizations to create:

  • A histogram showing the number of neutral sentiment datapoints for all airlines in the dataset

  • A histogram showing the count of each sentiment for the “Delta” airline

  • A separate pie chart for each airline, showing the sentiment percentage for each sentiment

  • Your choice: choose one additional visualization to create that shows something of interest to you using this dataset

Lastly, answer the following research questions based on the visualizations. Include your answers in your Jupyter notebook as Markdown.

Research Questions:

  • Which airlines are doing a good job capturing customer satisfaction?

  • How should American airlines strategically differentiate themselves in the coming year? What opportunities do they have?

  • How should Southwest airlines strategically differentiate themselves in the coming year? What opportunities do they have?

Self Driving Sentiment Data

Conduct some initial data exploration to become familiar with the dataset.

Then, begin visualizing the data in Jupyter notebook using Plotly.

Visualizations to create:

  • Scatter plot showing sentiment on the x axis, with the y axis indicating the level of confidence on the sentiment

  • Scatter plot indicating the length of the response (in characters) on the y axis and the confidence level on the x axis

  • Pie chart showing a summary of all sentiments, but only sentiment judgements with a confidence of higher than 0.8.

  • Your choice: choose one additional visualization to create that shows something of interest to you using this dataset

Lastly, answer the following research questions based on the visualizations. Include your answers in your Jupyter notebook as Markdown.

Research questions:

  • What are some of the primary concerns about self-driving cars, based on this dataset?

  • What is the public sentiment like for self-driving cars?
